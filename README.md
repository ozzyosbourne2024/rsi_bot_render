# RSI Bot – Render Scheduled Job

Bu klasör, RSI raporları ve alarm gönderimi için **Render üzerinde Scheduled Job** olarak çalıştırılacak script içerir.  
Script, belirlenen saatlerde çalışır ve RSI değerlerine göre Telegram’a alarm gönderir.

---

## 1️⃣ Gereksinimler

Python 3.10+ kurulu olmalı.  

`requirements.txt` dosyasındaki paketler yüklenecek:

```bash
pip install -r requirements.txt
